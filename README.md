# NetLogo
## What is NetLogo?
NetLogo is a programmable modeling environment for simulating natural and social phenomena. 
It is a powerful yet easy-to-learn environment for agent-based modeling in both research and education.

##### NOTE: NetLogo is a special-purpose Domain Specific Language; hence, it can only be used within the NetLogo environment. It cannot be run in IDEs like VS Code. Below are guidlines for how to install NetLogo and run a project within it.
